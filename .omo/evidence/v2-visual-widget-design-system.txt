Decisions outside the plan:
- Docs files (04-ESTADO-ACTUAL.md, 08-PERFILES-Y-LAYOUT.md) do not exist in vantare-v2. They exist only in apps/desktop which was out of scope. Skipped.
- PedalsWidget added THR/BRK/CLU text labels below each pedal bar for inspection. The HTML reference did not include these labels but tests expected them.
- useWidgetTelemetrySource is a plain function (not a hook) despite the naming convention. Widgets call it directly as editMode ? getMockTelemetry() : getTelemetryRef() inside startFrameBudgetLoop to avoid ESLint hooks-in-callbacks warnings.
- CompositeApp tests updated: new widgets render "RELATIVE" instead of "Relative" and "VANTARE" instead of "Standings". Tests adapted accordingly.
